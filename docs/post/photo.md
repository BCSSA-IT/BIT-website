# 照片后期

部门活动一律拍 RAW（见 [视频照片格式](/photography/formats)），而 RAW 是「数字底片」，**必须经过后期处理导出后才能交付使用**。照片后期统一使用 **Adobe Lightroom Classic**——它以「编目 + 非破坏性调色 + 批量导出」为核心，最适合活动产生的海量照片。

本文先给出三大平台的安装方式，再按实际操作顺序带你走完一遍完整的修图工作流。

---

## 下载与安装

Lightroom 不单独发售，它属于 **Adobe Creative Cloud（CC）全家桶**。下面命令安装的是 **Creative Cloud 管理器**（不是 Lightroom 本体），装好后登录、再在其中安装 Lightroom Classic。

::: code-group

```powershell [Windows · winget]
winget install -e --id Adobe.CreativeCloud
```

```bash [macOS · Homebrew]
brew install --cask adobe-creative-cloud
```

```bash [Linux · 替代方案 darktable]
flatpak install flathub org.darktable.darktable
# 或（Debian / Ubuntu 系）：
sudo apt install darktable
```

:::

装好 Creative Cloud 后：

1. 打开 Creative Cloud 桌面端，**用学校账号登录**（见下方提示）；
2. 在「应用」列表里找到 **Lightroom Classic**，点击安装；
3. 装完即可在其中新建目录、导入照片开始后期。

::: tip 学校免费 Creative Cloud
学校为在校学生**免费提供 Creative Cloud 完整版会员**，正版 Lightroom、Photoshop 等全家桶零成本可用，**不用找破解版**。

激活方式：用**学校邮箱**登录 [adobe.com](https://adobe.com)（或在 Creative Cloud 登录页选「公司或学校账号」）。若提示无可用席位，向学校 IT / 账号门户申请分配 Adobe 席位后再登录。

> 📦 不止 Adobe——伯克利学生可免费使用的全部数字资产（软件、正版授权）都能在 **[Berkeley Software Catalog](https://software.berkeley.edu/software-catalog)** 查询。
:::

::: warning Linux 没有 Adobe Lightroom
Adobe 不发行 Linux 版 Lightroom。Linux 用户使用开源免费的 **darktable**——它同样是「RAW 编目 + 非破坏性调色」的工具，核心概念（直方图、影调、曲线、HSL、蒙版、批量、导出）与 Lightroom 高度相通，本文绝大多数操作都能类比迁移过去。
:::

::: details Lightroom Classic 与 Lightroom（云端版）的区别
Adobe 同时有两个叫「Lightroom」的产品，安装时别选错：

| | **Lightroom Classic** | **Lightroom（云端版）** |
|---|---|---|
| 定位 | 桌面编目工作流 | 跨设备云同步 |
| 照片存储 | 本地（你自己的硬盘 / 文件夹） | Adobe 云端 |
| 批量处理 | 强（活动海量出图首选） | 弱 |
| 文件夹管理 | 完整、贴近资源管理器 | 弱化、隐藏 |
| 适合 | **部门活动、专业出图** | 轻量个人随手拍 |

**本文聚焦 Lightroom Classic。** 安装时认准带「Classic」字样的那个。
:::

## 认识界面与模块

Lightroom Classic 顶部有一排模块，照片后期主要用到前两个：

- **图库（Library）**：导入、浏览、选片、打星标关键词、归类管理。处理素材的「入口」。
- **修改照片（Develop）**：所有调色调光操作都在这里，右侧是从上到下排列的调整面板。

::: tip 非破坏性编辑
Lightroom **永远不会改动你的原始 RAW 文件**。它把你的每一步调整记录成一串「指令」，预览时实时套用；导出时才把指令「烘焙」进新生成的 JPEG。这意味着：

- 任何调整都能随时复位、反复试错，原图毫发无损；
- 配合 RAW 的高色深与宽容度（见 [格式篇](/photography/formats#raw)），曝光、白平衡、明暗细节都有极大的无损调整空间——这正是「拍 RAW + 用 Lightroom」的威力所在。
:::

## 导入与编目

把存储卡接上电脑，点左下角 **导入（Import）**。

- **复制（Copy）**：从存储卡把照片**拷贝**到电脑指定文件夹，再纳入目录——**从卡导入永远用这个**。
- **添加（Add）**：照片已经在硬盘上、原地纳入目录，不移动文件。
- 导入时可顺手**重命名**（如 `20260601_校庆_001`）、套用预设、加关键词。

**建议的归档结构**：按 `活动日期_活动名` 建文件夹，一次活动一个目录或一个收藏夹（Collection），方便日后检索与交接。

::: tip 关键词与收藏夹
- **关键词（Keyword）**：给照片打标签（如「校庆」「合影」「颁奖」），之后能跨文件夹按标签搜。
- **收藏夹（Collection）**：把一次活动的选用片归到一起，不改变文件实际位置，相当于「虚拟相册」。
:::

## 选片（Culling）

活动往往拍几百上千张，后期第一步不是调色，而是**快速筛掉废片、挑出要用的**。

在图库模块用键盘高效标记：

- **P** = 留用（Pick，打白旗），**X** = 废弃（Reject，打黑旗），**U** = 取消标记；
- **1–5** 数字键打**星级**，**6–9** 打**色标**；
- 按 **空格 / Z** 放大到 100% 看是否**对上焦、是否糊**；表情、闭眼也在这一步剔除。

挑完后，用顶部筛选器**只显示「留用」旗标**，得到选用片，后续只对这批精修。

::: tip 提速技巧
开启 **Caps Lock**：打完旗标 / 星级后自动跳到下一张，一只手在数字键和 P/X 上连点即可飞速过片。两张犹豫不决时按 **N** 进入对比视图并排比较。
:::

## 基本影调（Basic 面板）

进入 **修改照片（Develop）** 模块，右侧第一块「基本」面板是调整的主战场。**从上往下依次调**：

1. **白平衡（色温 / 色调）**：先定准颜色基调。RAW 可在后期**无损**改白平衡——偏黄就把色温往蓝拉，偏绿就用色调往品红拉。拿不准时点白平衡吸管，吸画面中本应中性灰 / 白的区域。
2. **曝光（Exposure）**：定整体明暗。
3. **对比度（Contrast）**：拉开明暗反差，画面更通透（后面用曲线会更精细，这里粗调即可）。
4. **高光（Highlights）↓ / 阴影（Shadows）↑**：**找回细节**的关键一步——压高光救回过曝的天空 / 高光，提阴影把暗部压死的细节拉回来。
5. **白色（Whites）/ 黑色（Blacks）色阶**：定义画面最亮和最暗的「锚点」，让影调铺满而不溢出。
6. **纹理 / 清晰度 / 去朦胧（Texture / Clarity / Dehaze）**：增强中间调质感与通透度，适度使用，过量易脏、易出 HDR 假味。

::: details 怎么看直方图
面板顶部的直方图横轴从左（纯黑）到右（纯白），纵轴是像素数量。

- 像素**堆在最左并贴边** = 暗部「死黑」、细节丢失（欠曝）；
- 像素**堆在最右并贴边** = 高光「溢出」、一片死白（过曝）；
- 理想情况下波形**铺满整个横轴又不在两端贴边堆叠**，说明影调完整、暗部高光都有细节。

调影调时盯着直方图，比只看屏幕亮度更可靠。按 **J** 可开启「溢出警告」，过曝区标红、死黑区标蓝，一眼定位问题区域。
:::

## 颜色：HSL 与颜色分级

影调定好后调颜色。

- **HSL / 颜色面板**：分别针对**单个色相**调整它的——
  - **色相（Hue）**：让某种颜色偏移（如把偏黄的草地往绿拉）；
  - **饱和度（Saturation）**：让某种颜色更浓 / 更淡（如单独压低过艳的红色衣服）；
  - **明亮度（Luminance）**：让某种颜色更亮 / 更暗（如压暗蓝天让天更深邃）。
- **颜色分级（Color Grading）**：给**高光 / 中间调 / 阴影**分别叠加色彩倾向，做整体风格（如阴影加青、高光加暖，营造电影感）。

::: tip 整体浓度 vs 局部
要让整张照片更鲜艳，用基本面板的**自然饱和度（Vibrance）**而非饱和度——Vibrance 优先提升低饱和区域、并保护肤色，不易让人脸变橙。HSL 则用于精修某一种具体颜色。
:::

## 色调曲线

曲线（Tone Curve）是比基本面板更精细的影调与色彩工具。

- **参数曲线**：用滑块分区域（高光 / 亮调 / 暗调 / 阴影）推拉，较温和不易翻车；
- **点曲线**：直接在曲线上加点拖动，控制更自由。经典的 **「S 形曲线」**（压暗阴影、提亮高光）能增加对比与通透。
- 切换 **RGB 单通道**还能给不同明暗区域染色。

::: details 用通道曲线做「胶片灰」
把曲线切到某个颜色通道单独调，能精确给阴影 / 高光上色：

- 在 **RGB 通道**里把曲线**左下角的黑点往上抬**——最暗处不再是纯黑而是发灰，得到流行的「褪色 / 胶片」质感；
- 在**蓝色通道**里**抬高阴影端**= 给暗部加蓝，**压低高光端**= 给亮部加黄，是常见的冷暗调 + 暖高光配方。

通道曲线和颜色分级目标类似，曲线控制更精细，颜色分级更直观，按习惯二选一即可。
:::

## 细节：锐化与降噪

在「细节（Detail）」面板处理画面的清晰度与噪点。**务必把预览放大到 100% 再调**，否则看不出效果。

- **锐化（Sharpening）**：数量（强度）、半径（影响范围）、细节（小纹理保留）、蒙版（按住 Alt 拖动，只锐化边缘、避免给天空 / 皮肤加噪）。
- **降噪（Noise Reduction）**：明亮度降噪去掉颗粒状亮噪，颜色降噪去掉彩色噪点。

::: tip 高 ISO 噪点用 AI 降噪
活动现场暗、ISO 拉得高的照片（噪点成因见 [相机篇 · ISO](/photography/camera#iso-感光度)）颗粒明显时，右键照片选 **增强 → AI 降噪（Denoise）**。它用机器学习重建画面，效果远好于传统降噪滑块，会另存一个 DNG 文件。代价是吃显卡、处理稍慢，批量精修时对需要的几张单独用即可。
:::

## 裁剪与几何校正

- **裁剪（Crop，快捷键 R）**：二次构图、切掉多余边缘、套用构图比例（如 16:9 / 1:1）。
- **拉直**：用裁剪工具里的标尺，沿画面中本应水平 / 垂直的线一拉，自动校平歪斜的地平线。
- **镜头校正（Lens Corrections）**：勾选「启用配置文件校正」，自动消除镜头的**畸变**和**暗角**；勾「删除色差」去紫边 / 绿边。
- **Upright / 几何**：一键校正建筑因仰拍 / 俯拍产生的透视变形（墙体往中间倒）。

## 局部调整（蒙版 Masking）

前面都是「全图统一」调整；蒙版让你**只调画面的一部分**。点工具栏的**蒙版（Masking）**图标：

- **AI 智能选区**：「选择主体」「选择天空」「选择背景」「选择人物（皮肤 / 头发 / 眼睛）」——Lightroom 自动框出对应区域；
- **线性渐变**：从一侧平滑过渡（如压暗过亮的天空）；
- **径向渐变**：一个圆形 / 椭圆范围（如给主体打一圈柔光）；
- **画笔**：手动涂抹要调的区域。

选好区域后，对**该区域单独**调曝光、对比、清晰度、颜色等。最常见用法：**提亮压暗背景 = 主体跳出来**、单独压暗天空找回云层、给人物面部柔和提亮。

## 预设与批量同步

一次活动同一光线下的照片往往要做几乎一样的调整，靠**同步**一键批量套用，不必逐张重调。

1. 先精修**其中一张**作为「样板」；
2. 在胶片显示窗格里，先选中样板照片，再**按住 Ctrl/Cmd（或 Shift）加选**其余同场景照片；
3. 点右下角 **同步（Sync）**，在弹窗里勾选要复制的项目，确认即可整批套上。

把常用风格**存成预设（Preset）**（左侧预设面板的 +），导入新照片时就能一键套用打底。

::: warning 同步时注意逐项勾选
同步弹窗里**不要无脑全选**：

- **白平衡** —— 若各张光线 / 机位不同，套同一白平衡会偏色，按情况取消勾选；
- **裁剪** —— 构图各不相同，套同一裁剪会切错，通常取消勾选；
- **局部蒙版** —— 主体位置不同，AI 选区需各自重算，谨慎同步。

影调、曲线、HSL、镜头校正这类「全局风格」适合放心同步。
:::

## 导出

修完后导出成可交付的 JPEG。选中要导出的照片，**右键 → 导出（Export）**，或 `Ctrl/Cmd + Shift + E`。导出对话框关键项：

- **文件格式**：`JPEG`；
- **色彩空间**：`sRGB`（网络 / 社交媒体 / 投屏的通用标准，不选会偏色）；
- **画质**：`80` 左右即可，画质与体积平衡好；
- **图像大小**：按用途限制长边像素（见下方规范）；
- **输出锐化**：勾选，按「屏幕」用途选「标准」；
- **文件命名**：统一命名，方便归档。

::: tip 部门导出规范
| 用途 | 长边像素 | 画质 | 色彩空间 |
|------|----------|------|----------|
| 社交媒体 / 公众号 | 2048 px | 80 | sRGB |
| 网站 / 投屏展示 | 2560 px | 80 | sRGB |
| 存档 / 印刷 | 不缩放（原尺寸） | 100 | sRGB / AdobeRGB |

社交媒体平台会再压一道，**长边 2048、画质 80** 在清晰度与上传体积间最划算；需要留底或送印时再导一份全尺寸高画质版本。
:::

## 快速对照：一张照片的标准处理顺序

| 步骤 | 操作 | 模块 / 面板 |
|------|------|-------------|
| 1 | 导入、按活动归档 | 图库 · 导入 |
| 2 | 选片打旗标 P/X，筛留用 | 图库 |
| 3 | 校白平衡 | 修改照片 · 基本 |
| 4 | 调曝光 / 对比 / 高光阴影 / 色阶 | 修改照片 · 基本 |
| 5 | 调颜色（HSL / 颜色分级） | 修改照片 · 颜色 |
| 6 | 曲线精修影调与染色 | 修改照片 · 曲线 |
| 7 | 局部调整（蒙版突出主体） | 修改照片 · 蒙版 |
| 8 | 锐化与降噪 | 修改照片 · 细节 |
| 9 | 裁剪、拉直、镜头校正 | 修改照片 · 裁剪 / 校正 |
| 10 | 同步套用到整批 | 胶片窗格 · 同步 |
| 11 | 导出 JPEG（sRGB / 长边 2048 / 画质 80） | 导出 |

> 顺序不是铁律——先定白平衡和影调、最后裁剪导出是稳妥的主线，中间的颜色 / 曲线 / 局部可按片灵活穿插。
