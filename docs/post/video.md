# 视频后期

部门活动视频拍完（统一 1080p / 60fps，可能是 Log，见 [视频照片格式](/photography/formats#视频格式)），都要经过后期才能交付：**剪辑**（留好的、去废的、定节奏）、**调色**（校正 + 风格）、**加字幕音乐**，最后**导出**成可发布的成片。

剪辑软件不止一个——先按自己的情况选对工具，再上手基本功能。本文不深挖某一个软件，重点是帮你**选**，并把各软件的**基本功能**讲清楚够用。

---

## 先选对软件

::: tip 第一原则：用你已经最熟的那个
手里要是有用顺手的剪辑软件，**就用它**——熟练度带来的效率远比换新工具高。下面的推荐只针对「还没有特别熟的软件」的情况。
:::

没有特别趁手的，按你的目标挑：

| 你的情况 | 推荐 | 为什么 |
|----------|------|--------|
| 纯新手 / 想快出片 | **剪映 CapCut** | 上手最快，模板、自动字幕、音乐一应俱全 |
| 想学专业 / 长期做 | **Premiere Pro（+After Effects）** | 行业标准；伯克利学生免费 |
| 想要顶级调色 / 要免费 | **DaVinci Resolve** | 免费版就极强，调色是业界天花板 |
| **Linux 用户** | **DaVinci Resolve** | 剪映 / PR / AE 都没有 Linux 版，只能用它 |

::: tip 伯克利免费 Adobe
Premiere Pro、After Effects 属于 Creative Cloud 全家桶，**伯克利学生免费使用**。安装与领取方式见[照片后期 · 下载与安装](/post/photo#下载与安装)。
:::

## 剪映 CapCut — 新手首选

模板化、傻瓜式，最快出片。活动快剪、社媒短视频的首选。

::: code-group

```powershell [Windows · winget]
winget install -e --id ByteDance.CapCut
```

```bash [macOS · Homebrew]
brew install --cask capcut
```

```text [Linux]
# 剪映 / CapCut 没有 Linux 版 → 用下方的 DaVinci Resolve
```

:::

**基本功能**

- **导入素材**：把视频/图片拖进媒体区，再拖到下方时间线。
- **剪辑**：分割（剃刀）、删除、拖动调整顺序与时长。
- **转场**：在两个片段之间拖一个转场效果即可。
- **自动字幕**：识别语音**一键生成字幕**——剪映最香的功能之一，省去手打。
- **音乐 / 音效**：内置曲库与音效，支持踩节奏卡点。
- **滤镜 / 特效 / 模板**：一键套用现成风格与花字。
- **导出**：选 1080p 一键导出成片。

::: tip 适合什么
活动快剪、Vlog、公众号 / 小红书 / 抖音短视频。需要精细调色或复杂动效时，再转 PR 或达芬奇。
:::

## Premiere Pro + After Effects — 专业 & 伯克利免费

**Premiere Pro（PR）** 是主力剪辑软件；**After Effects（AE）** 负责动效、片头与字幕动画，两者可「动态链接」配合。行业标准工具，且伯克利免费。

::: warning 绝对强大，但门槛高
PR / AE 功能极其强大、**几乎能满足一切需求**，代价是**上手难度高**，而且很多效果要靠**第三方插件**堆出来（转场、字幕动画、调色预设等，下方的马头人就是其一）。若还要做 **3D 场景 / 三维元素**，通常得再搭配 **Blender**（免费开源）建模渲染，再合成回 AE。只想快速出片就先用剪映；确定要深耕影视后期，再上这套。
:::

**安装**：PR、AE 都通过 **Creative Cloud** 安装（用学校账号登录后在 CC 里装），步骤见[照片后期 · 下载与安装](/post/photo#下载与安装)。

**Premiere Pro 基本功能**

- **项目与序列**：新建项目，按素材的分辨率 / 帧率建序列（1080p 60fps）。
- **时间线剪辑**：剃刀（`C`）切片段、波纹删除、拖动排序，是剪辑主战场。
- **转场**：视频过渡（交叉溶解等）拖到片段衔接处。
- **Lumetri 调色**：基础校色（曝光 / 白平衡 / 对比 / 饱和），Log 素材在这里**套 LUT** 还原（呼应 [格式篇 · Log](/photography/formats#log-模式)）。
- **基本图形（Essential Graphics）**：做字幕、标题、下沿条。
- **音频**：多轨音频 + 基础混音。
- **导出**：直接导出或送 Media Encoder，输出 H.264 / MP4。

**After Effects** 简单说：用「合成 + 关键帧」做片头、字幕动效、简单特效，做好后通过动态链接放回 PR。

::: tip 关键帧（Keyframe）是动效的基础
**关键帧**就是「在某个时间点，给某个属性记下一个值」；在两个关键帧之间，软件自动补出中间过渡，画面就「动」起来了。比如第 0 秒不透明度记 0、第 1 秒记 100，就得到一段淡入；位置、缩放、旋转、音量同理。PR、AE、达芬奇乃至剪映都靠它做动画，是后期最该先掌握的概念之一——**先理解关键帧，再谈动效**。
:::

::: tip 马头人免费插件（Premiere Composer）
**马头人（Mister Horse）** 出的 **Premiere Composer** 是 PR 的免费扩展：先装它家的「**Mister Horse Product Manager**」，在里面安装 Premiere Composer，自带的 **Starter Pack 免费**就能解锁一批**转场、文字动画 / 文字框、音效**，拖进时间线即用——新手快速出效果非常方便。官网 [misterhorse.com/premiere-composer](https://misterhorse.com/premiere-composer)。

> AE 侧对应的免费插件是 **Animation Composer**，同样来自马头人，做动效时可一并装上。
:::

## DaVinci Resolve — 免费强力 & Linux 唯一选择

免费版几乎不阉割，**调色**是它的王牌；也是 **Linux 上唯一能打的**专业剪辑软件。

::: code-group

```powershell [Windows · winget]
winget install -e --id BlackmagicDesign.DaVinciResolve
```

```text [macOS · App Store]
# 在 Mac App Store 搜索 "DaVinci Resolve"，免费下载安装
```

```bash [Linux · 官网]
# 官网下载（官方支持 Rocky/CentOS，Ubuntu 等亦可安装），需免费注册
# https://www.blackmagicdesign.com/products/davinciresolve
```

:::

**页面式工作流**（底部切换页面，每个页面对应一个环节）：

- **Cut / Edit**：剪辑（快速粗剪 / 精细剪辑）。
- **Fusion**：节点式特效与动画。
- **Color**：调色——一级 / 二级校色、节点、示波器，**它的核心卖点**。
- **Fairlight**：音频混音。
- **Deliver**：导出。

**基本流程**：素材导入媒体池 → Edit 页时间线剪辑 → Color 页调色（Log 素材用色彩管理 / LUT）→ Deliver 页导出 H.264 / H.265 的 MP4。

::: tip 免费版够用吗？
够。免费版已覆盖剪辑 / 调色 / 音频 / 导出全流程；付费的 **Studio** 才解锁部分 AI 工具、降噪、超 60fps 等，**部门日常用免费版完全足够**。
:::

## 通用剪辑流程

不管用哪个软件，活动视频后期大致都是同一条主线：

1. **整理导入** — 素材归类、导入软件。
2. **粗剪** — 留好的、去废的、把顺序排顺。
3. **精剪** — 抠节奏、加转场，让片子流畅。
4. **调色** — 先校正（曝光 / 白平衡），再定风格；Log 素材套 LUT（见 [格式篇](/photography/formats#log-模式)）。
5. **音频** — 对白清晰、配乐音效、整体响度统一。
6. **字幕 / 图形** — 字幕、标题、下沿条。
7. **导出** — 按下方规范输出。

## 导出规范

::: tip 部门导出规范 · 视频
| 项目 | 推荐 |
|------|------|
| 分辨率 / 帧率 | 1080p / 60fps（跟拍摄一致；纯发布也可 30fps） |
| 编码 | H.264（最兼容）或 H.265（省体积） |
| 封装 | MP4 |
| 码率 | 1080p 约 16–24 Mbps |
| 音频 | AAC，约 320 kbps |

社媒平台上传后会再压一道，按上表导出能在清晰度与体积间取得最优；需要留底 / 二次剪辑时另存一份高码率版本。
:::
